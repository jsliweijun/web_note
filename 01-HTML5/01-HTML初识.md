# HTML初识

html 是 文本标签语言。超文本标记语言。

体会关键词：文本，标签，语言。

**文本**（使用文本实现，非图片，视频），但如果只是文本没有作用。

**标签**（有具体的格式，<> 左右尖括号），文本形成标签后，起到对页面特殊效果的作用。

**语言**（是语言就有语法，有规范，有结构）。

html：用文本表示，带有标签的语言，作用于网页框架的搭建实现。



网页有什么组成的： 文本，图片，音频，视频 。 这些元素组成的。

这些元素都是 用 html 标签进行表示。



html基本骨架

```javascript
<!DOCTYPE html>
<html>
	<head>
    	<title></title>
    </head>
	<body></body>
</html>
```



标签分类： 双标签 <></> ，单标签 < />



单标签,数量不多，书写时，字母与 / 有空格是标准写法。

| 标签       | 作用 |
| ---------- | ---- |
| `<meta />` |      |
| `<base />` |      |
| `<hr />`   |      |
| `<img />`  |      |
| `<br />`   |      |
