# 对象 Object

对象是一直复合数据类型，里面可包含多种基本数据类型的数据。可把对象当做是一个容器。

对象分类：

* 内建对象：es标准提供的。Math，String，Boolean，Function，Object。
* 宿主对象：浏览器提供的，window，document，console
* 自定义对象：开发人员创建的。



## 对象的基本操作

```javascript
var obj = new Object();
// 直接写 Object() 就是调用一个函数。
// new Object()  new 后面调用的是构造函数，构造函数专门使用来创建对象的。
```
